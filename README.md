# 🦇 Dark Horoscope

Proyecto de aprendizaje desarrollado como repaso de JavaScript y Tailwind, inspirado en un ejercicio propuesto por [Codédex](https://www.codedex.io). El ejercicio original era mucho más sencillo, pero me encanta complicarme la vida.

## ¿Qué hace?

El usuario introduce su fecha de nacimiento y un murciélago oracular le revela su signo del zodíaco y una fortuna aleatoria con mucha actitud.

## ¿Qué he practicado?

- Condicionales y control de flujo para calcular el signo según mes y día
- Manipulación del DOM con `querySelector`, `textContent` y `classList`
- Eventos con `addEventListener` y `preventDefault`
- Generación de números aleatorios con `Math.random()`
- Manipulación de strings con `split()` y template literals
- Efecto de escritura letra a letra con `setInterval`

## Tecnologías

- JavaScript vanilla
- HTML5
- Tailwind CSS v4
- Vite

## Instalación

```bash
npm install
npm run dev
```

## Nota

Este proyecto forma parte de mi proceso de aprendizaje de desarrollo frontend. El ejercicio original de Codédex proponía una versión básica por meses; la versión final incluye cálculo por fecha exacta, interfaz completa y un murciélago gótico con bocadillo animado porque las cosas hay que hacerlas bien.
